## 项目概述

面向家庭的旅游规划 H5 网页应用。查询国内景点和天气，智能生成个性化旅游路线，自动学习家人偏好（白名单/黑名单），实现越用越懂你的推荐体验。

## 技术选型

| 项目 | 选型 | 说明 |
|------|------|------|
| 前端 | 纯 HTML/CSS/JS | 移动端优先，无需框架 |
| 后端 | Python Flask | 前后端一体部署 |
| 存储 | JSON 文件 | 偏好和配置轻量存储 |
| 景点 & 天气 | 高德地图 API | 一个 Key 搞定全部 |
| 部署 | Render（免费） | Flask 直接服务前端静态文件 |

## 核心功能

### ✅ 已实现

| 功能 | 说明 |
|------|------|
| 🔍 景点搜索 | 按关键词 + 城市搜索，显示名称/地址/类型/评分/门票估算 |
| 🌤️ 天气查询 | 输入城市查看 4 天预报，含温度/风力/天气状况 |
| 🗺️ 路线生成 | 输入目的地+天数+预算+出发地，自动规划每日行程，**每天显示天气** |
| 💰 预算管控 | 自动估算交通/住宿/餐饮/门票，对比预算给出提示 |
| ⭐ 白名单 | 喜欢的景点/类型优先推荐，路线中 👍 一键加入 |
| 👎 黑名单 | 不喜欢的自动过滤，路线中 👎 一键加入 |
| 🏙️ 默认城市 | 设置常驻城市，自动作为出发地 |
| 📱 移动端 | 4 个底部导航标签，手机上直接使用 |
| 🌐 公网部署 | Render 免费部署，家人随时随地访问 |

### 🔄 智能推荐机制

```
路线生成流程：
  搜索景点 → 过滤黑名单（名称+类型） → 白名单加分排序 → 预算约束 → 
  天数分配 → 获取天气 → 输出每日行程+费用+天气
```

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/health` | 健康检查 |
| GET | `/api/attractions?keyword=&city=` | 搜索景点 |
| GET | `/api/weather?city=` | 查询天气 |
| POST | `/api/route/generate` | 生成路线（含天气） |
| GET | `/api/preferences` | 获取偏好的白名单/黑名单 |
| POST | `/api/preferences` | 添加偏好 |
| DELETE | `/api/preferences/<id>` | 删除偏好 |
| GET | `/api/config` | 获取配置 |
| PUT | `/api/config` | 更新配置 |
